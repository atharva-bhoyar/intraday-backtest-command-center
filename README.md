# Intraday Trading Strategy Backtesting & Command Center

This project implements a Python-based intraday trading strategy backtest and a Streamlit dashboard to visualize performance.

## Features
- Intraday OHLCV data processing
- Moving Average based trading strategy
- PnL, drawdown, and daily returns calculation
- Automated report generation
- Interactive Streamlit dashboard

## Tech Stack
- Python
- Pandas, NumPy
- Streamlit
- Plotly
- Jupyter Notebook

## How to Run
1. Run the Jupyter notebook to generate reports:
   - `backtest_project.ipynb`
2. Start the dashboard:
   ```bash
   python -m streamlit run src/app.py
